# Altofalante — Guia de Marca

## Essência
**Nome:** Altofalante
**Tagline:** _"Seu celular, sua caixa de som."_
**Promessa:** som mais alto e mais limpo, sem hardware extra — e, junto com outros
celulares, um sistema de som de verdade.
**Personalidade:** energética, moderna, direta, sem frescura. Fala como amigo, não como
manual técnico.

## Logo
- **Marca (ícone):** alto-falante estilizado + ondas sonoras, dentro de um quadrado
  arredondado com gradiente. Arquivos: `app-icon.svg` / `app-icon.png`.
- **Lockup horizontal:** ícone + wordmark `alto` (claro) **`falante`** (rosa).
  Arquivo: `logo.svg` / `logo.png`.
- **Área de proteção:** mantenha ao redor do logo um espaço livre = altura da letra "a".
- **Não faça:** distorcer, trocar as cores do gradiente, colocar sobre fundos claros sem
  contraste, ou adicionar sombras pesadas.

## Cores
| Token | Hex | Uso |
|-------|-----|-----|
| `bg` | `#0A0A0F` | fundo principal (tema escuro) |
| `surface` | `#16161F` | cartões / chips inativos |
| `surface-2` | `#1F1F2B` | elevação / divisores |
| `primary` | `#A855F7` | violeta (início do gradiente) |
| `primary-2` | `#EC4899` | rosa (fim do gradiente) — cor de marca |
| `accent` | `#22D3EE` | ciano — destaques (loudness, multi-celular) |
| `text` | `#F5F5FA` | texto principal |
| `muted` | `#8B8B99` | texto secundário |

**Gradiente da marca:** `#A855F7 → #D946C6 → #EC4899` (135°). Use no botão Turbo, no
ícone e em elementos de destaque. O ciano é o "tempero" — use com parcimônia.

## Tipografia
- **Títulos / wordmark:** sans geométrico, peso 700–800 (sugestão: **Sora** ou
  **Outfit** via Google Fonts no Flutter).
- **Texto:** **Inter** ou a fonte de sistema, peso 400–600.
- Letras com leve _letter-spacing_ negativo em títulos grandes.

## Tom de voz
- Curto e animado. "Bota no Turbo." / "Som no talo, sem chiado."
- Honesto: nunca prometer virar uma JBL; prometer "o máximo que seu celular dá".
